# roadie-dotnetcore


roadie-dotnetcore
======
Powerful API Music Server.

Works well with roadie-vuejs and also has a full [Subsonic compatible API](http://www.subsonic.org/pages/apps.jsp) that works with many Subsonic mobile applications. Roadie was built to be able to handle music collections with [hundreds of thousands of songs](http://www.redferret.net/?page_id=38781).
